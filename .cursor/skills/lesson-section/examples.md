# Section Outline Template

After completing the interview, produce this single structured outline. Only include blocks for which the trainer provided answers.

```markdown
## Section outline: [Section title]

### 1. section

- Title: [exact section title]

### 2. why

- Key message: [one sentence]
- Bullet points / narrative: [2–4 short points from trainer]

### 3. little-what

- One-sentence primer: [exact sentence(s) from trainer]

### 4. Content slides (in order)

- [Layout]: [short description of content, bullets or example reference]
- [Layout]: [Include any: default, sub-section, two-cols, two-cols-header, image-left, image-right, code example refs]

### 5. Visual for visual learners

- Description: [what to draw or which image to use]
- Suggested layout: image-left | image-right | default (Mermaid)

### 6. Optional

- task: [brief task description if given]
- what-if: [brief extension/question if given]
- ask-me-anything: [prompt text if given]
```
