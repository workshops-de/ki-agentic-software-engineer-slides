# What if you could give the model knowledge it was never trained on?

That's exactly what **Retrieval-Augmented Generation (RAG)** does — inject fresh, relevant documents into the context at query time. The model's pattern-matching applies to your data, not just its training set.

# What if the model could call tools, browse the web, or run code?

That's the **agentic paradigm** — LLMs are no longer just text predictors. They become orchestrators that decide when to call APIs, execute code, or search the internet. Understanding token prediction is the foundation for understanding why agents still fail, hallucinate, and lose context.
