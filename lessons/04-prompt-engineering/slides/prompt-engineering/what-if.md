# What if prompting itself becomes automated?

Models are getting better at understanding intent with less instruction — where does that leave prompt engineering?

---
layout: two-cols-header
layoutClass: gap-4
---

# The Evolving Landscape

::left::

**Now–2 years**

- Persistent memory reduce repetitive context-setting
- IDEs auto-construct prompts from files and changes
- Prompt templates become assets in version control
- Evaluation of output quality matters more

::right::

**Longer term (5+ years)**

- Models infer intent with minimal instruction
- Prompting shifts from writing to **reviewing AI proposals**
- The skill moves from _crafting prompts_ to _evaluating output_

::bottom::

<Callout type="important">
The underlying skill — clear, structured thinking — will be valuable no matter how AI interfaces evolve.
</Callout>
